# Plant Disease Prediction 🌿

This project is an AI-powered plant disease prediction system that detects diseases in plant leaves using deep learning. The model analyzes leaf images and predicts the type of disease affecting the plant. This helps farmers and researchers identify plant diseases early and take appropriate action to protect crops.

The application is built using TensorFlow for the deep learning model and Streamlit for creating an interactive web interface. Users can upload an image of a plant leaf, and the system will process the image and display the predicted disease.

## Features
- Upload plant leaf images
- Detect plant diseases using a trained deep learning model
- Simple and interactive web interface using Streamlit
- Fast prediction results

## Technologies Used
- Python
- TensorFlow
- Streamlit
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## How to Run the Project

1. Install the required libraries
pip install -r requirements.txt

2. Run the  application
streamlit run main.py

3. Open the provided local URL in your browser to use the app.
